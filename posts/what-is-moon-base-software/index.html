What is Moon Base Software?

I'm starting Moon Base Software as an open source project to provide a highly abstract data model can represent any type of data you can imagine. Having everything in a common model allows data of many different types to be combined and compared, and visualised together. This project also aims to build visualisations that can used across multiple platforms, and build useful apps building on this data model and visualisations.
